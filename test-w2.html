Single Responsive Image

CSS ONLY

Without using @media query the images still resize to the varying screensize.


BOOTSTRAP ONLY

The image will automaticall resize to the varying screensize.

Multiple Responsive Images

The responsive images (since it's class is img-fluid) will be placed next to each other depending on the column width specified in your class and will appear on top of each other on smaller screens.
